# Features in simple language

## Live tiles
Live tiles show changing app information on the Start screen. They help the user see calls, messages, calendar events, weather, and diagnostics without opening each app. Internally, apps publish small signed tile records to the compositor service.

## Lock screen
The lock screen protects the phone from accidental use. It displays time, notifications, battery, and emergency call access. It works by keeping applications paused until the user unlocks with the configured credential.

## Compatibility layers
XAP and APPX packages are inspected, unpacked, permission-checked, and launched inside sandboxes when their APIs are implemented. EXE support is limited to technically possible ARMv7 PE programs through open Wine/ReactOS-style APIs. APK support is isolated in an Android-compatible runtime boundary and does not claim universal compatibility.

## Cellular capability
The Lumia 625H hardware supports GSM, EDGE, UMTS, HSPA, HSPA+, and LTE through its modem generation. It does not support 5G, so Bolun Phone OS reports no 5G capability instead of pretending.
